# TPT-48Ch-CVC 電池監控系統

這是一個 48 通道電池電壓和電流監控系統，具有圖形化使用者介面。

## 功能特色

- 即時監控 48 個電池通道的電壓和電流
- 圖形化介面顯示每個通道的狀態
- 根據電壓範圍自動調整通道顏色顯示
- 每 15 秒自動更新數據
- 支援自動均衡和強制均衡功能
- 顯示總電壓、總電流等摘要資訊

## 檔案說明

- `old/48ch_gui.py` - 主要的 GUI 應用程式
- `old/main.py` - 命令列版本的數據產生器
- `old/icon.ico` - 應用程式圖示
- `old/TPT-48Ch-CVC.exe` - 編譯後的可執行檔

## 系統需求

- Python 3.x
- tkinter (通常隨 Python 安裝)

## 使用方法

### 執行 GUI 版本
```bash
python old/48ch_gui.py
```

### 執行命令列版本
```bash
python old/main.py
```

## 介面說明

- **通道網格**: 顯示 48 個通道的電壓值，顏色代表不同狀態
  - 紅色: 通道 1 (特殊標示)
  - 黃色: 通道 9 (特殊標示)
  - 淺藍色: 電壓低於 3.100V
  - 淺綠色: 電壓高於 4.150V
  - 白色: 正常範圍

- **狀態面板**: 顯示系統狀態和參數
- **控制按鈕**: 自動均衡和強制均衡功能
- **摘要面板**: 顯示最高/最低電壓等統計資訊
